Чтобы установить PostgreSQL, выполните следующие действия.

1. Откройте терминал WSL (Internet Explorer). Ubuntu 18,04).
2. Обновите пакеты Ubuntu: `sudo apt update`
3. После обновления пакетов установите PostgreSQL (и пакет от участников сообщества с некоторыми полезными служебными программами) с помощью: `sudo apt install postgresql postgresql-contrib`.
4. Подтвердите установку и получите номер версии: `psql --version`

Есть три команды, которые необходимо знать после установки PostgreSQL:

1. `sudo service postgresql status` для проверки состояния базы данных.
2. `sudo service postgresql start`, чтобы начать работу с базой данных.
3. `sudo service postgresql stop` для завершения работы базы данных.

### <a name="postgresql-user-setup"></a>Настройка пользователя PostgreSQL

Пользователь с правами администратора по умолчанию, `postgres`, должен иметь пароль, назначенный для подключения к базе данных. Чтобы задать пароль, сделайте следующее:

1. Введите команду: `sudo passwd postgres`.
2. Появится запрос на ввод нового пароля.
3. Закройте и снова откройте терминал.

### <a name="run-postgresql-with-psql-shell"></a>Запуск PostgreSQL с помощью оболочки psql

[psql](https://www.postgresql.org/docs/10/app-psql.html) — это внешний интерфейс на основе терминала для PostgreSQL. Он позволяет интерактивно вводить запросы, выдавать их в PostgreSQL и просматривать результаты запроса. Кроме того, входные данные могут быть из файла. Кроме того, он предоставляет ряд мета-команд и различные функции, аналогичные оболочке, для упрощения написания сценариев и автоматизации самых разнообразных задач.

Чтобы запустить оболочку PSQL, выполните следующие действия.

1. Запустите службу postgres: `sudo service postgresql start`
2. Подключитесь к службе postgres и откройте оболочку psql: `sudo -u postgres psql`

После успешного входа в оболочку psql вы увидите, что ваша командная строка выглядит следующим образом: `postgres=#`.

> [!NOTE]
> Кроме того, можно открыть оболочку PSQL, перейдя к пользователю postgres с помощью команды: `su - postgres` и введя команду: `psql`.

Чтобы выйти из postgres = # Enter: `\q` или использовать сочетание клавиш: Ctrl + D

Чтобы узнать, какие учетные записи пользователей были созданы в установке PostgreSQL, используйте WSL Terminal: `psql -c "\du"`... или просто `\du`, если открыта оболочка psql. Эта команда будет отображать столбцы: Имя пользователя учетной записи, список атрибутов ролей и член групп ролей. Чтобы выйти из командной строки, введите: `q`.
