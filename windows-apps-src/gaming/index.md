Дополнительные сведения о разработке игр UWP с использованием DirectX см. в разделе [Программирование для DirectX](directx-programming.md).

Для максимально эффективного изучения обзоров и учебников по разработке игр ознакомьтесь с перечисленными ниже темами.

-   Microsoft C++ с расширениями компонентов (C++/CX). Это обновление Microsoft C++, в которое входит автоматический подсчет ссылок, является языком разработки игр UWP с DirectX 11.1 или более поздних версий.
-   Базовое знание терминологии программирования графики.
-   Базовые навыки программирования в Windows.
-   Базовые знания об интерфейсах API Direct3D 9 или Direct3D 11.
