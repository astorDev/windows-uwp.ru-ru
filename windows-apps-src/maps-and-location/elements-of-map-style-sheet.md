# <a name="map-style-sheet-reference"></a>Справка о таблицах стилей карты

Использование технологий Microsoft сопоставления _сопоставить стилей_ для определения внешнего вида карт.  Карта стилей определяется с помощью нотации объектов JavaScript (JSON) и может использоваться в различных способов, включая в приложении Windows Store [MapControl](https://docs.microsoft.com/uwp/api/windows.ui.xaml.controls.maps.mapcontrol) через [MapStyleSheet.ParseFromJson](https://docs.microsoft.com/uwp/api/windows.ui.xaml.controls.maps.mapstylesheet.parsefromjson#Windows_UI_Xaml_Controls_Maps_MapStyleSheet_ParseFromJson_System_String_) метод.

Таблицы стилей могут создаваться в интерактивном режиме с помощью [редактор таблиц стилей карты](https://www.microsoft.com/p/map-style-sheet-editor/9nbhtcjt72ft) приложения.

Приведенный ниже код JSON можно использовать чтобы воды отображаются красным цветом, воды метки отображаться зеленым цветом и land области отображаются синим цветом:

```json
    {"version":"1.*",
        "settings":{"landColor":"#0000FF"},
        "elements":{"water":{"fillColor":"#FF0000","labelColor":"#00FF00"}}
    }
```

Этот код JSON можно использовать для удаления всех меток и точек с карты.

```json

    {"version":"1.*", "elements":{"mapElement":{"labelVisible":false},"point":{"visible":false}}}
```

Иногда для получения окончательного результата значение свойства преобразуется.  Например, fillColor растительности имеет немного различающиеся оттенки в зависимости от типа отображаемого объекта.  Это поведение можно отключить, используя точно указанное значение, с помощью свойства ignoreTransform.

```json
    {"version":"1.*",
        "settings":{"shadedReliefVisible":false},
        "elements":{"vegetation":{"fillColor":{"value":"#999999","ignoreTransform":true}}}
    }
```

В этой статье показаны записи JSON и [свойства](#properties), которые можно использовать для настройки внешнего вида карт.  Эти свойства могут также применяться к элементам карты пользователя с использованием [MapStyleSheetEntry](https://docs.microsoft.com/uwp/api/windows.ui.xaml.controls.maps.mapelement.mapstylesheetentry) свойство.
