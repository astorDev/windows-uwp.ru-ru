# <a name="camera"></a>Камера

Этот раздел содержит рекомендации по созданию приложений универсальной платформы Windows (UWP), которые используют камеру или микрофон для захвата фотографий, видео или аудио.

## <a name="use-the-windows-built-in-camera-ui"></a>Использование встроенного пользовательского интерфейса камеры в Windows

| Раздел | Описание |
|---------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Захват фото и видео со встроенной камерой Windows пользовательского интерфейса](capture-photos-and-video-with-cameracaptureui.md) | Здесь рассказывается, как использовать класс [**CameraCaptureUI**](https://docs.microsoft.com/uwp/api/Windows.Media.Capture.CameraCaptureUI) для фото- и видеосъемки с помощью встроенного пользовательского интерфейса камеры в Windows. Если требуется просто предоставить пользователю возможность снимать фотографии или видеоклипы и возвращать результат в приложение, это самый быстрый и простой способ.  |

| [Запись последовательности фотографии с VariablePhotoSequence](variable-photo-sequence.md) | Здесь рассказывается, как захватить переменную серию фотографий (то есть несколько снимков, быстро следующих один за другим) и настроить для каждого снимка различные параметры фокусировки, вспышки, числа ISO, экспозиции и коррекции экспозиции.  |
