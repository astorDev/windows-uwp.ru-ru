# <a name="uwp-app-streaming-install"></a>Потоковая установка приложений UWP
Потоковая установка приложений универсальной платформы Windows (UWP) позволяет указать Microsoft Store, какие части вашего приложения следует скачивать в первую очередь. Когда основные файлы приложения будут загружены, пользователь сможет запустить приложение и работать с ним, пока остальные файлы будут скачиваться в фоновом режиме. 

Для использования UWP потоковая передача Установка приложения вам потребуются разделить файлы ваше приложение на разделы. Для этого вы создадите карта контента группы, которая XML-файла, упакованный в приложении, позволяя порядок и приоритет загрузки набора. В разделе связанные ниже для получения дополнительных сведений.

Полное руководство по добавлению приложения UWP UWP потоковая передача Установка приложения найти в этой [серии блогов](https://blogs.msdn.microsoft.com/appinstaller/2017/03/15/uwp-streaming-app-installation/).
