# <a name="accessibility"></a>Специальные возможности  

Специальные возможности в том, что приложение универсальная платформа Windows (UWP) может использоваться людьми, использующими технологию в широком спектре сред, и подходом пользовательского интерфейса с разнообразными потребностями и опытом. В некоторых ситуациях специальные возможности требуются законодательством. Однако рекомендуется реализовывать специальные возможности даже там, где они не обязательны по закону, так как это расширит аудиторию вашего приложения.
