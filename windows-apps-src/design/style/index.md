# <a name="style-for-uwp-apps"></a>Стиль приложений UWP

Стиль помогает выделить ваше приложение среди других. Вам нужно принять смелые эстетические решения и добавить элегантные украшения. Главное— создать что-то красивое. 

## <a name="style-and-the-fluent-design-system"></a>Стиль и система проектирования Fluent Design

<p>Система проектирования Fluent Design позволяет создавать современные и эффективные пользовательские интерфейсы, которые отличаются яркостью, глубиной, движением, материальностью и масштабированием. Рекомендации в этом разделе помогут создать стиль Fluent Design, используя цвет, шрифтовое оформление и новые функции, например <a href="../style/acrylic.md">акриловый материал</a> и <a href="../style/reveal.md">эффект отображения</a>. 
</p>
