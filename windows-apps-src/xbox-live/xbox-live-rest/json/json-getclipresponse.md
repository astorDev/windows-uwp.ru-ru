# <a name="getclipresponse-json"></a>GetClipResponse (JSON)
Создает оболочку для игр клипа. 
<a id="ID4EN"></a>

 
## <a name="getclipresponse"></a>GetClipResponse
 
Объект GetClipResponse имеет следующей спецификации.
 
| Участник| Тип| Описание| 
| --- | --- | --- | 
| GameClip| [GameClip](json-gameclip.md)| Один игр клип, удовлетворившего операцию запроса.|
