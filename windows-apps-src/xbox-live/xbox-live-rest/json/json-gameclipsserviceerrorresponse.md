## <a name="gameclipsserviceerrorresponse"></a>GameClipsServiceErrorResponse
 
Объект GameClipsServiceErrorResponse имеет следующей спецификации.
 
| Участник| Тип| Описание| 
| --- | --- | --- | 
| <b>d, источник ошибки</b>| Строка| Источник ошибки.|
