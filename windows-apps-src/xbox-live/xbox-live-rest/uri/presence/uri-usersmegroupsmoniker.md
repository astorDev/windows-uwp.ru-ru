# <a name="usersmegroupsmoniker"></a>/users/me/groups/{moniker}
Обращается к PresenceRecord нашей группе. Доменом для таких URI является `userpresence.xboxlive.com`.
 
  * [Параметры URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Параметры универсального кода ресурса (URI)
 
| Параметр| Тип| Описание| 
| --- | --- | --- | 
| моникер| Строка| Строка, определяющая группу пользователей. Только принятые моникер в настоящее время является «Люди», с заглавной буквы «P».| 
  
<a id="ID4ERB"></a>

 
## <a name="valid-methods"></a>Допустимые методы

[Получение (/ users/me/группы / {моникер})](uri-usersmegroupsmonikerget.md)

&nbsp;&nbsp;Получает PresenceRecord для моей группы.
