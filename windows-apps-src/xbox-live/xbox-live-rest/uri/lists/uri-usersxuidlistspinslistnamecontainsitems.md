# <a name="usersxuidxuidlistspinslistnamecontainsitems"></a>/users/xuid(xuid)/lists/PINS/{listname}/ContainsItems
Определяет, содержатся ли в наборе элементов (как указано в itemId) в списке без получения весь список. Доменом для таких URI является `eplists.xboxlive.com`.
 
  * [Параметры URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Параметры универсального кода ресурса (URI) 
 
| Параметр| Тип| Описание| 
| --- | --- | --- | 
| XUID| Строка| XUID пользователя.| 
| ListName| Строка| Имя списка для управления.| 
  
<a id="ID4E5B"></a>

&nbsp;&nbsp;Определяет, содержит ли список в наборе элементов (как указано в itemId) без получения весь список.
