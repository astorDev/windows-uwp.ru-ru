# <a name="usersxuidxuidlistspinslistnameremoveitems"></a>/users/xuid(xuid)/lists/PINS/{listname}/RemoveItems
Удаляет элементы из списка. Доменом для таких URI является `eplists.xboxlive.com`.
 
  * [Параметры URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Параметры универсального кода ресурса (URI) 
 
| Параметр| Тип| Описание| 
| --- | --- | --- | 
| XUID| Строка| XUID пользователя.| 
| ListName| Строка| Имя списка для управления.| 
  
<a id="ID4E5B"></a>

 
## <a name="valid-methods"></a>Допустимые методы

[УДАЛЕНИЕ](uri-usersxuidlistspinslistnameremoveitemsdelete.md)

&nbsp;&nbsp;Удаляет элементы из списка по индексу.

[ПОМЕСТИТЬ](uri-usersxuidlistspinslistnameremoveitemspost.md)

&nbsp;&nbsp;Удаляет элементы из списка, itemId.
