# <a name="privacy-uris"></a>Коды URI конфиденциальности
 
В этом разделе представлены сведения о универсальный идентификатор ресурса (URI) адреса и связанных методов протокола передачи гипертекста (HTTP) от службы Xbox Live для *конфиденциальности*.
 
Только игровых программ для Windows 8, на Xbox 360 или Windows Phone устройство может использовать эту службу.
 
Доменом для таких URI является privacy.xboxlive.com.
