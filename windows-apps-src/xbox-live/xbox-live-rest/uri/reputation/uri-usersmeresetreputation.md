# <a name="usersmeresetreputation"></a>/users/me/resetreputation
Позволяет группе принудительного применения для доступа к оценки репутации текущего пользователя. Доменом для таких URI является `reputation.xboxlive.com`.
  
<a id="ID4EV"></a>

 
## <a name="valid-methods"></a>Допустимые методы

[POST (/ пользователи/me/resetreputation)](uri-usersmeresetreputationpost.md)

&nbsp;&nbsp;Позволяет команде разработчиков применение оценки репутации текущего пользователя некоторые произвольные значения после равным (к примеру) захват учетной записи.
