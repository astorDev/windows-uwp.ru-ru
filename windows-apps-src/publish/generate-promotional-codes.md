# <a name="generate-promotional-codes"></a>Создание промокодов


[Центр партнеров](https://partner.microsoft.com/dashboard) позволяет создавать Промокоды для приложения или надстройки, опубликованную в Microsoft Store. Рекламные коды — это простой способ предоставить влиятельным пользователям бесплатный доступ к вашему приложению или надстройке. Можно также использовать Промокоды для сценариев клиентов службы, предоставив пользователям приложения или надстройки, или для бесплатного доступа [бета-тестирования](beta-testing-and-targeted-distribution.md) с Windows 10. 

Каждый код акции имеет соответствующий предъявителю уникальный URL, в который клиента можно щелкнуть для этого кода и установки приложения или надстройки из Microsoft Store.  Ваше приложение должно пройти заключительный этап публикации в [процессе сертификации приложения](the-app-certification-process.md), прежде чем клиенты смогут использовать рекламный код для его установки.

Можно создавать коды однократного использования (и передать его для каждого клиента), или можно выбрать для создания кода, который может использоваться несколько раз с заданным числом клиентов.

> [!TIP]
> Вы можете использовать [целевые push-уведомления](send-push-notifications-to-your-apps-customers.md), чтобы предоставлять рекламные коды определенному сегменту пользователей. При этом обязательно используйте рекламный код, который могут использовать несколько клиентов.


## <a name="promotional-code-policies"></a>Политики рекламных кодов

Для рекламных кодов можно использовать указанные ниже политики.

-   Можно создавать рекламные коды для любого приложения или надстройки (кроме надстроек с подпиской), опубликованных в Microsoft Store. Клиенты могут использовать коды во всех версиях ОС Windows, поддерживаемых приложением или надстройкой.
-   Срок действия рекламных кодов истекает через 6 месяцев после даты их заказа (если не была выбрана более ранняя дата окончания срока действия).
-   Для каждого приложения или надстройки можно создавать коды, допускающие до 1600 случаев использования каждые 6 месяцев. Шестимесячный период начинается при отправке первого заказа на рекламный код, даже если вы выберете более раннюю дату окончания срока его действия. Общее количество случаев использования (1600) для каждого продукта относится как к разовым кодам, так и тем, которые можно использовать несколько раз.
-   Необходимо следовать требованиям, заданным в [соглашение с разработчиком приложений](https://docs.microsoft.com/legal/windows/agreements/app-developer-agreement), включая раздел **3 k. Промокоды**.

> [!NOTE]
> Можно использовать поощрительные коды, даже если ваше приложение будет недоступна для клиентов (то есть, в том случае, если вы выбрали **сделать этот продукт доступен, но не обнаруживается в Store** с **остановить приобретения: Любой клиент с прямой ссылкой можно увидеть список Store продукта, но их можно только загрузить его, если они принадлежит продукт до, или код акции и вы используете устройство Windows 10** параметр в вашу заявку [возможность обнаружения ](choose-visibility-options.md#discoverability) раздел). Этот параметр клиенты должны иметь в Windows 10 (включая Xbox), чтобы получить продукт с рекламный код.


## <a name="order-promotional-codes"></a>Заказ рекламных кодов

Для заказа Промокоды для приложения или надстройки:

1.  В меню навигации слева [центра партнеров](https://partner.microsoft.com/dashboard), разверните **Attract** , а затем выберите **Промокоды**.

2.   На странице **Рекламные коды** щелкните **Заказать коды**.

3.  На странице **Новый заказ рекламных кодов** введите указанные ниже сведения.
    -   Выберите приложение или надстройку, для которых необходимо создать коды. (Обратите внимание, что создать промокоды для надстроек с подпиской невозможно.)
    -   Укажите имя заказа. Вы можете использовать это имя, чтобы отличать заказы на коды при изучении данных об использовании рекламных кодов.
    -   Выберите тип заказа. Можно создать набор промокодов, каждый из которых можно использовать один раз, или создать один промокод, который можно использовать несколько раз.
    -   Укажите количество кодов в заказе (при создании набора кодов) или количество раз, когда можно использовать код (при создании одного кода для многоразового использования).
    -   Укажите, когда рекламные коды должны активироваться. Чтобы выбрать определенную дату и время, снимите флажок **Коды активируются немедленно**. В противном случае кодов станет активным прямо сейчас (несмотря на то, что ваш продукт необходимо выполнить процесс публикации в заказ для клиента с использованием кода).
    -   Укажите, когда истекает срок действия рекламных кодов. Чтобы выбрать определенную дату и время истечения срока действия (ранее 6 месяцев), снимите флажок **Срок действия кодов истекает через 6 месяцев**.

4.  Щелкните **Заказать коды**. После этого вы вернетесь на страницу **Рекламные коды**, где сможете увидеть новый заказ в сводной таблице заказов на рекламные коды для этого приложения.


## <a name="download-and-distribute-promotional-codes"></a>Загрузка и распространение рекламных кодов

Чтобы скачать выполненный заказ на рекламные коды и распределить коды среди клиентов, выполните указанные ниже действия.

1.  В меню навигации слева [центра партнеров](https://partner.microsoft.com/dashboard), разверните **Attract** , а затем выберите **коды акции.**
2.  Перейдите по ссылке **Скачать** для вашего заказа на рекламный код и сохраните созданный файл на компьютер. Этот файл содержит информацию о вашем заказе на рекламные коды в виде значений, разделенных знаками табуляции (TSV).
3.  Откройте TSV-файл в удобном для вас редакторе. Для большего удобства откройте TSV-файл в приложении, которое может отображать данные в виде таблицы, например в Microsoft Excel. Однако файл можно также открыть в любом текстовом редакторе.

    Файл содержит указанные ниже столбцы данных для каждого кода.

    -   **Название продукта**: Имя приложения или надстройки, с которой связан код.
    -   **Имя порядка**: Имя порядка, в котором этот код был создан.
    -   **Код акции**: Сам код. Это строка алфавитно-цифровых символов, разделенная дефисами на пять групп по пять символов в каждой группе. Например: DM3GY-M2GYM-6YMW6-4QHHT-23W2Z
    -   **Предъявителю URL-адрес**: URL-адрес, который клиент может использовать для этого кода и установки приложения или надстройки. URL-адрес имеет следующий формат: https://go.microsoft.com/fwlink/?LinkId=532540&mstoken=&lt; promotional_code >
    -   **Дата начала**: Дата, когда этот код стали активными.
    -   **Срок действия даты**: Дата истечения срока действия этого кода.
    -   **Код идентификатора**: Уникальный идентификатор для этого кода.
    -   **Идентификатор заказа**: Уникальный идентификатор для заказа, в котором этот код был создан.
    -   **Присвоенный**: Пустое поле, можно использовать для отслеживания клиенте, который вы присвоили кода для.
    -   **Доступно:** Количество раз, когда код по-прежнему доступен активировать (во время создания файла).
    -   **Обменять**: Количество раз, которые активировал код (во время создания файла).

4.  Вы можете передавать URL-адреса с промокодом вашим клиентам с помощью любых каналов связи (например, в виде целевых push-уведомлений, по электронной почте, в SMS-сообщениях или на печатных карточках). При предоставлении кода пользователям мы рекомендуем сообщать им указанные ниже сведения.
    -   Сведения о том, для какого приложения или надстройки предназначен рекламный код, а также необязательное описание того, почему клиент получает этот код.
    -   Погашаемый URL-адрес для кода.
    -   Инструкции для клиента о том, как использовать URL-адрес с промокодом, войти в систему с помощью учетной записи Майкрософт и следовать указаниям по скачиванию и установке вашего приложения.


## <a name="code-redemption-user-experience"></a>Взаимодействие с пользователем при использовании кода

После предоставления рекламный код (или его предъявителю URL-адрес) клиента, необходимо щелкнуть URL-адрес бесплатного продукта. Щелчок по погашаемому URL-адресу приведет к открытию страницы **Использование кода** с проверкой подлинности в <https://account.microsoft.com/billing/redeem>. Эта страница включает описание приложения, которое пользователь собирается получить. Если пользователь не выполнил вход с помощью своей учетной записи Майкрософт, ему может быть предложено сделать это. Ваш клиент также можно посетить <https://account.microsoft.com/billing/redeem> и ввести код напрямую.

> [!IMPORTANT]
> Мы не рекомендуем предоставлять рекламные коды клиентам, пока не будет завершен процесса публикации продукта (даже если вы выбрали вариант **Сделать этот продукт доступным, но невидимым в Магазине**). При попытке использовать рекламный код продукта, который еще не был опубликован, клиентам отобразится ошибка.

После того как клиент нажмет кнопку **Использовать**, откроется страница обзора приложения в Microsoft Store (если клиент использует устройство под управлением ОС Windows 10 или Windows 8.1), где пользователь может нажать кнопку **Установить**, чтобы скачать и установить приложение бесплатно. Если клиент использует компьютер или устройство, на которых не установлен Microsoft Store, по ссылке откроется веб-страница Microsoft Store для вашего приложения. Код будет применяться к учетной записи Майкрософт клиента, чтобы он мог позже скачать приложение бесплатно на устройство с Windows (связанное с той же учетной записью Майкрософт).

> [!NOTE]
> В некоторых случаях может появиться клиента **купить** вместо кнопки **установить**, несмотря на то, что приложение было успешно активированы с помощью льготного кода. Клиент может нажать кнопку **Купить**, чтобы бесплатно установить приложение.


## <a name="review-your-promotional-codes"></a>Просмотр сведений о рекламных кодах

Чтобы просмотреть подробную сводку заказов специальный код для приложения и надстройки, перейдите к **Промокоды** страницы (в левом навигационном меню центра партнеров, разверните **Attract** , а затем выберите **Промокоды**). Вы можете просмотреть указанные ниже сведения о всех текущих и неактивных рекламных кодах для приложения.
-   Имя заказа
-   Приложение или надстройка
-   Дата начала
-   Дата истечения срока действия
-   Доступно.
-   Использован

Кроме того, из этой таблицы можно [скачать](#download-and-distribute-promotional-codes) заказ.
