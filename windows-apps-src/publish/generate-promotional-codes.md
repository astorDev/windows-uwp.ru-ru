# <a name="generate-promotional-codes"></a>Создание промокодов


Вы можете создавать рекламные коды для приложения или надстройки, опубликованных в Microsoft Store. Рекламные коды — это простой способ предоставить влиятельным пользователям бесплатный доступ к вашему приложению или надстройке. Можно также использовать рекламные коды в сценариях обслуживания клиентов, предоставляя пользователям бесплатный доступ вашего приложения или надстройки, или для [бета-тестирования](beta-testing-and-targeted-distribution.md) с помощью Windows10. 

Каждый рекламный код имеет соответствующий уникальный погашаемый URL-адрес, клиент может щелкнуть использовать код и установить приложение или надстройку из Microsoft Store.  Ваше приложение должно пройти заключительный этап публикации в [процессе сертификации приложения](the-app-certification-process.md), прежде чем клиенты смогут использовать рекламный код для его установки.

Можно создавать коды однократного использования (и передать его для каждого клиента), или вы можете создать код, который можно использовать несколько раз, заданное число клиентов.

> [!TIP]
> Вы можете использовать [целевые push-уведомления](send-push-notifications-to-your-apps-customers.md), чтобы предоставлять рекламные коды определенному сегменту пользователей. При этом обязательно используйте рекламный код, который могут использовать несколько клиентов.


## <a name="promotional-code-policies"></a>Политики рекламных кодов

Для рекламных кодов можно использовать указанные ниже политики.

-   Можно создавать рекламные коды для любого приложения или надстройки (кроме надстроек с подпиской), опубликованных в Microsoft Store. Клиенты могут использовать коды во всех версиях ОС Windows, поддерживаемых приложением или надстройкой.
-   Срок действия рекламных кодов истекает через 6 месяцев после даты их заказа (если не была выбрана более ранняя дата окончания срока действия).
-   Для каждого приложения или надстройки можно создавать коды, допускающие до 1600случаев использования каждые 6месяцев. Шестимесячный период начинается при отправке первого заказа на рекламный код, даже если вы выберете более раннюю дату окончания срока его действия. Общее количество случаев использования (1600) для каждого продукта относится как к разовым кодам, так и тем, которые можно использовать несколько раз.
-   Вы должны соблюдать требования, указанные в [соглашении с разработчиком приложений](https://docs.microsoft.com/legal/windows/agreements/app-developer-agreement), включая раздел **3k. Рекламные коды**.

> [!NOTE]
> Можно использовать рекламные коды, даже если ваше приложение было недоступно клиентам (то есть, в том случае, если вы выбрали **сделать этот продукт доступным, но невидимым в магазине** с **Остановить приобретение: любой пользователь, у которых может увидеть продукта в магазине Описание, но они могут скачать его, только если они владел раньше или имеет промокод и использует устройство под управлением Windows 10** параметр в разделе [возможность обнаружения](choose-visibility-options.md#discoverability) вашей отправки). С помощью этого параметра пользователи должны находиться в Windows 10 (включая Xbox), чтобы приобрести продукт с рекламным кодом.


## <a name="order-promotional-codes"></a>Заказ рекламных кодов

Чтобы заказать рекламные коды для приложения или надстройки:

1.  В левом меню навигации на информационной панели Центра разработки для Windows разверните узел **Привлечение**, а затем выберите **Промокоды**.

2.   На странице **Рекламные коды** щелкните **Заказать коды**.

3.  На странице **Новый заказ рекламных кодов** введите указанные ниже сведения.
    -   Выберите приложение или надстройку, для которых необходимо создать коды. (Обратите внимание, что создать промокоды для надстроек с подпиской невозможно.)
    -   Укажите имя заказа. Вы можете использовать это имя, чтобы отличать заказы на коды при изучении данных об использовании рекламных кодов.
    -   Выберите тип заказа. Можно создать набор промокодов, каждый из которых можно использовать один раз, или создать один промокод, который можно использовать несколько раз.
    -   Укажите количество кодов в заказе (при создании набора кодов) или количество раз, когда можно использовать код (при создании одного кода для многоразового использования).
    -   Укажите, когда рекламные коды должны активироваться. Чтобы выбрать определенную дату и время, снимите флажок **Коды активируются немедленно**. В противном случае коды станут сразу же (несмотря на то, что необходимо выполнить процесс публикации в порядке, клиент может использовать код продукта).
    -   Укажите, когда истекает срок действия рекламных кодов. Чтобы выбрать определенную дату и время истечения срока действия (ранее 6месяцев), снимите флажок **Срок действия кодов истекает через 6месяцев**.

4.  Щелкните **Заказать коды**. После этого вы вернетесь на страницу **Рекламные коды**, где сможете увидеть новый заказ в сводной таблице заказов на рекламные коды для этого приложения.


## <a name="download-and-distribute-promotional-codes"></a>Скачивание и распространение рекламных кодов

Чтобы скачать выполненный заказ на рекламные коды и распределить коды среди клиентов, выполните указанные ниже действия.

1.  В левом меню навигации на информационной панели Центра разработки для Windows разверните узел **Привлечение**, а затем выберите **Промокоды**.
2.  Перейдите по ссылке **Скачать** для вашего заказа на рекламный код и сохраните созданный файл на компьютер. Этот файл содержит информацию о вашем заказе на рекламные коды в виде значений, разделенных знаками табуляции (TSV).
3.  Откройте TSV-файл в удобном для вас редакторе. Для большего удобства откройте TSV-файл в приложении, которое может отображать данные в виде таблицы, например в MicrosoftExcel. Однако файл можно также открыть в любом текстовом редакторе.

    Файл содержит указанные ниже столбцы данных для каждого кода.

    -   **Название продукта**: имя приложения или надстройка, с которыми связан код.
    -   **Название заказа**: название заказа, в котором был создан этот код.
    -   **Рекламный код**: сам код. Это строка алфавитно-цифровых символов, разделенная дефисами на пятьгрупп по пятьсимволов в каждой. Например, так: DM3GY-M2GYM-6YMW6-4QHHT-23W2Z.
    -   **URL-адрес с промокодом**: URL-адрес, с помощью которого клиент может использовать код и установить ваши приложения или надстройки. URL-адрес имеет следующий формат: http://go.microsoft.com/fwlink/?LinkId=532540&mstoken=&lt; promotional_code >
    -   **Дата начала срока действия**: дата, когда этот код стал активным.
    -   **Дата истечения срока действия**: дата истечения срока действия кода.
    -   **Идентификатор кода**: уникальный идентификатор кода.
    -   **Идентификатор заказа**: уникальный идентификатор заказа, в котором был создан этот код.
    -   **Предоставлен пользователю**: пустое поле, которое можно использовать для учета того, какому клиенту вы предоставили код.
    -   **Доступен**: количество раз, когда код по-прежнему доступен для использования (во время создания файла).
    -   **Использован**: количество раз, когда был использован код (во время создания файла).

4.  Вы можете передавать URL-адреса с промокодом вашим клиентам с помощью любых каналов связи (например, в виде целевых push-уведомлений, по электронной почте, в SMS-сообщениях или на печатных карточках). При предоставлении кода мы рекомендуем сообщать указанные ниже сведения.
    -   Сведения о том, для какого приложения или надстройки предназначен рекламный код, а также необязательное описание того, почему клиент получает этот код.
    -   URL-адрес с промокодом для этого кода.
    -   Инструкции для клиента о том, как использовать URL-адрес с промокодом, войти в систему с помощью учетной записи Майкрософт и следовать указаниям по скачиванию и установке вашего приложения.


## <a name="code-redemption-user-experience"></a>Взаимодействие с пользователем при использовании кода

Получив рекламный код (или его погашаемый URL-адрес) для клиента, можно щелкнуть URL-адрес, чтобы получить продукт бесплатно. Щелчок по погашаемому URL-адресу приведет к открытию страницы **Использование кода** с проверкой подлинности в <https://account.microsoft.com/billing/redeem>. Эта страница содержит описание приложения, которое пользователь собирается использовать. Если пользователь не выполнил вход с помощью своей учетной записи Майкрософт, ему может быть предложено сделать это. Ваш клиент также можно посетить <https://account.microsoft.com/billing/redeem> и ввести код напрямую.

> [!IMPORTANT]
> Мы не рекомендуем предоставлять рекламные коды клиентам, пока не будет завершен процесса публикации продукта (даже если вы выбрали вариант **Сделать этот продукт доступным, но невидимым в Магазине**). При попытке использовать рекламный код продукта, который еще не был опубликован, клиентам отобразится ошибка.

После того как клиент нажмет кнопку **Использовать**, откроется страница обзора приложения в Microsoft Store (если клиент использует устройство под управлением ОС Windows10 или Windows8.1), где пользователь может нажать кнопку **Установить**, чтобы скачать и установить приложение бесплатно. Если клиент использует компьютер или устройство, на которых не установлен Microsoft Store, по ссылке откроется веб-страница Microsoft Store для вашего приложения. Код будет применяться к учетной записи Майкрософт клиента, чтобы он мог позже скачать приложение бесплатно на устройство с Windows (связанное с той же учетной записью Майкрософт).

> [!NOTE]
> В некоторых случаях клиент может см. в разделе кнопки **купить** вместо, **Установка**, несмотря на то, что приложение успешно приобретено с помощью рекламного кода. Клиент может нажать кнопку **Купить**, чтобы бесплатно установить приложение.


## <a name="review-your-promotional-codes"></a>Просмотр сведений о рекламных кодах

Чтобы просмотреть подробную сводку по заказам на рекламные коды для ваших приложений или надстроек, перейдите на страницу **Рекламные коды** (в левом меню навигации информационной панели Центра разработки), разверните пункт **Привлечение внимания** и выберите вариант **Рекламные коды**). Вы можете просмотреть указанные ниже сведения о всех текущих и неактивных рекламных кодах для приложения.
-   Название заказа
-   Приложение или надстройка
-   Дата начала срока действия
-   Дата истечения срока действия
-   Доступен
-   Использован

Кроме того, из этой таблицы можно [скачать](#download-and-distribute-promotional-codes) заказ.
