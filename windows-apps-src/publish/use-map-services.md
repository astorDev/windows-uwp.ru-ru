# <a name="use-map-services"></a>Использование служб карт

Чтобы использовать службы карт в приложениях, предназначенных для Windows 10 или Windows 8.x,, посетите [Центр разработки для Карт Bing](http://go.microsoft.com/fwlink/p/?LinkId=614880). Сведения о том, как запросить ключ карт проверки подлинности в центре разработчика Bing Maps и добавление его в приложении разделе [запроса ключа проверки подлинности карт](../maps-and-location/authentication-key.md) для получения дополнительных сведений. 

Для использования служб карты в приложениях для Windows Phone 8.1 и более ранних версий, необходимо запросить идентификатор приложения службы карты и маркер для включения в коде приложения в панели мониторинга центра разработки.

Любое время после добавили [зарезервированное имя вашего приложения](create-your-app-by-reserving-a-name.md), можно перейти к этого приложения в панели мониторинга, разверните раздел **служб** в меню навигации слева и выберите **карт**. При нажатии кнопки **получить маркер**, мы будем создать картографической службе идентификатор приложения (**идентификатор приложения**) и отображение службы проверки подлинности маркеров (**AuthenticationToken**) для вашего приложения. Эти значения будут сохранены на этой странице, чтобы можно было вернуться в любое время, чтобы просмотреть их.

> [!IMPORTANT]
> Необходимо добавить код перед группировать **ApplicationID** и **AuthenticationToken** и отправка ваше приложение. Дополнительные сведения см. в разделе [Добавление элемента управления картой на странице (Windows Phone 8.1)](http://go.microsoft.com/fwlink/p/?LinkId=614882).
