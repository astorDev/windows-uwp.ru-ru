# <a name="whats-new-in-the-windows-developer-docs-in-may-2018"></a>Что нового в документации для разработчиков Windows в мая 2018 г.

### <a name="msix-packaging-format"></a>Формат упаковки MSIX

Объявили на конференции Microsoft Build 2018 MSIX — это новый формат пакета значительное упрощает использование контейнеров, которая применяется для всех приложений Windows, включая Win32, Windows Forms, WPF и UWP. Этот новый формат UWP наследует функции:

* Надежной установки и обновления. 
* Управлять модель безопасности с системой гибкие возможности.
* Поддержка Microsoft Store, управление корпоративной и многие модели пользовательских распространения.

Средства для создания эти пакеты будут доступны в будущих выпусках Visual Studio и пакета SDK для Windows.

Формат упаковки MSIX — это формат с открытым исходным кодом, что упрощает наших партнеров для поддержки в экосистеме MSIX с помощью средства и решения. Для получения дополнительных сведений о формате MSIX упаковки, см. в разделе [MSIX SDK](https://github.com/Microsoft/msix-packaging).
