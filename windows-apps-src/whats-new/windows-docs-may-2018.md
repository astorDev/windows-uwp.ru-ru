### <a name="msix-packaging-format"></a>Создание пакета, содержащего формат MSIX

Объявлена на конференции Microsoft построения 2018 MSIX — это новый формат пакета containerization, которое применяется ко всем приложениям Windows, включая Win32, Windows Forms, WPF и UWP. Этот новый формат наследует от UWP обладает возможностями:

* Надежная установки и обновления. 
* Управляемая модель безопасности с системой гибкие возможности.
* Поддержка хранилища Майкрософт, Управление корпоративным информационным и многие модели рассылки.

Средства для создания эти пакеты будут доступны в будущих версиях Visual Studio и пакет SDK для Windows.

Создание пакета, содержащего MSIX используется формате открытым кодом, который упрощает партнеров для поддержки экосистеме MSIX с помощью своих средств и решений. Для получения дополнительных сведений о формате MSIX упаковки, найти [MSIX в пакете SDK](https://github.com/Microsoft/msix-packaging).
