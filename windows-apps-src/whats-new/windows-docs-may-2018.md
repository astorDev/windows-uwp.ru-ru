### <a name="msix-packaging-format"></a>Формат упаковки MSIX

Было объявлено на конференции Microsoft Build 2018, MSIX — это новый формат пакета контейнеризации, который применяется ко всем приложениям Windows, включая Win32, Windows Forms, WPF и UWP. Этот новый формат наследует отличные функции от универсальной платформы Windows:

* Надежной установки и обновления. 
* Управляемые модели безопасности с системой гибкие возможности.
* Поддержка Microsoft Store, управления предприятием и множество моделей нестандартного распределения.

Инструменты для создания этих пакетов будут доступны в будущих версиях Visual Studio и пакета SDK для Windows.

Формат упаковки MSIX — это формат открытым исходным кодом, что упрощает для наших партнеров для поддержки в экосистеме MSIX с их средств и решений. Дополнительные сведения о формате MSIX упаковки, см. в разделе [MSIX SDK](https://github.com/Microsoft/msix-packaging).
