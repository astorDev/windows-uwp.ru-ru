# <a name="stream-output-view-sov"></a>Представление выходного потока (SOV)


С помощью представления выходного потока данные о вершинах, которые были получены от шейдеров вершин, тесселяции и геометрии, можно передать потоком обратно в приложение для дальнейшего использования. Например, объект, который был обработан этими шейдерами, можно записать обратно в приложение для предоставления более точных входных данных физическому или другому модулю. Однако, представления выходного потока это редко используемая на практике функция графического конвейера.
