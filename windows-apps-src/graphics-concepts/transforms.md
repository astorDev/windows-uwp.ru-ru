# <a name="transforms"></a>Преобразования


Компонент Direct3D, который двигает геометрию через геометрический конвейер с фиксированными функциями, называется модулем преобразований. Он определяет положение модели и наблюдателя в мире, проецирует вершины для отображения на экране, и кадрирует вершины в соответствии с окном просмотра. Модуль преобразований также выполняет вычисления освещения для определения компонентов рассеяния и отражения в каждой вершине.
